# 💫 About Me:

Especially anything involving simulation, data pipelines, or real-time systems that don’t fall apart under load.<br><br>🤝 I’m looking for help with<br>Advanced backend architecture patterns, production-grade system design, and making complex infrastructure not feel like a punishment to maintain.<br><br>🌱 I’m currently learning<br>Deep dives into Rust async ecosystems, database internals (Postgres, Redis, vector extensions), and how data actually flows cleanly from backend to frontend without becoming spaghetti.<br><br>💬 Ask me about<br>Flight dynamics concepts, Rust backend design, GraphQL vs tRPC tradeoffs, or why most “scalable” systems aren’t actually scalable.<br><br>⚡ Fun fact<br>I design entire systems in my head before writing a single line of code… and then spend hours debugging why reality disagrees.
